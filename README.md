# 🏓 Ping-Pong-Game

> *Космический пинг-понг на выживание*

---

## 🌌 Лор

Вы — **капитан космического корабля**.  
Ваша задача — **очистить путь от космических стен** с помощью новейшего орудия — **плазменного мяча**.

💥 Мяч уничтожает любую стену одним касанием.  
⚠️ Но если мяч уйдёт от вас — начальство вас накажет. **Потерять мяч нельзя.**

---

## 🎯 Цель игры

Набрать **максимальное количество очков**.

> 📊 Данные прохождения автоматически сохраняются в `static.sqlite`

---

## 📜 Правила

- Игра продолжается, **пока мяч не коснётся нижней стенки**
- При касании **любой поверхности** мяч отлетает в противоположную сторону
- **Новый уровень** начинается сразу после уничтожения последней разрушимой стенки
- При старте нового уровня **игрок и мяч возвращаются в исходную позицию**

---

## 🧱 Создание своих уровней

Вы можете **сами создавать уровни** — для этого достаточно заполнить текстовый файл.

### 📐 Как это работает

Каждый символ в файле задаёт стенку.  
Сдвиг между стенками:

- по оси **X** — `50` пикселей
- по оси **Y** — `40` пикселей

### 🎨 Условные обозначения

| Символ | Цвет стены | Разрушимость |
|--------|------------|---------------|
| `G` | 🟢 зелёная | разрушимая |
| `Y` | 🟡 жёлтая | разрушимая |
| `R` | 🔴 красная | разрушимая |
| `B` | 🔵 синяя | разрушимая |
| `#` | ⚫ серая | **неразрушимая** |
| ` ` (пробел) | — | пустой символ (пропуск стены) |
