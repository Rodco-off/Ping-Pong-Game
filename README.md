# Ping-Pong-Game


### `Лор`

Вы капитан космического корабля, вам нужно очистить путь от космических стен.
Для этого вам нужно использовать новейшее орудие — плазменный мяч, который одним касанием уничтожает стенку.
Если мяч уйдёт от вас, ваше начальство вас накажет, так что вам нельзя его терять.

### `Цель игры`

Набрать как можно больше счёта.  
P.S. Данные прохождения загружаются в `static.sqlite`.

**Правила просты:**

- Играешь, пока мяч не коснётся нижней стенки;
- Мяч отлетает в противоположную сторону при касании любой поверхности;
- Новый уровень начинается сразу после того, как исчезнет последняя разрушимая стенка;
- Новый уровень начинается с перемещения игрока и мяча в исходную позицию.

Вы сами можете создавать свои уровни. Для этого надо заполнить текстовый файл.  
Каждый новый символ передвигает стенку на 50 пикселей по оси абсцисс и на 40 по оси ординат.

**Условные обозначения для создания уровня:**

| Символ | Значение |
|--------|----------|
| `G` | зелёная стена |
| `Y` | жёлтая стена |
| `R` | красная стена |
| `B` | синяя стена |
| `#` | неразрушимая стена серого цвета |
| пробел | пустой символ (пропуск стены) |

Также название текстового файла, в котором хранится ваш уровень, нужно написать в классе `GamePole` в атрибуте `LEVELS`.
